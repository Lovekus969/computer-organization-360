# computer-organization-360

Study notes, examples, and small tools for learning the IBM System/360 family concepts and classic computer organization topics.

> Focus: Instruction Set Architecture (ISA), CPU & memory organization, registers, addressing modes, I/O, and hands-on examples.

## Repo structure
- `docs/` — written notes and explanations (ISA, memory, CPU, I/O, history)
- `src/` — small code examples and simple simulators/assemblers
- `slides/` — presentation slides and lecture PDFs
- `exercises/` — practice problems and solutions
- `assets/` — diagrams and images
